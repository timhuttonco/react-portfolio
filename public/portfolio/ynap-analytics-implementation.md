### Senior Manager, Analytics Implementation at YOOX Net-A-Porter
#### August 2021 - Present

In my current role, I am responsible for managing the Analytics Implementation team at <a href="https://www.ynap.com/" target="_blank" title="YOOX Net-A-Porter">YOOX Net-A-Porter</a>, the world leader in online luxury and fashion.

I am incredibly fortunate to manage a talented team of Implementation Analysts and Architects, split across our London, Milan and Bologna offices, who sit inside the Growth function of the business.

The function is responsible for the implementation and alignment of data layers across all of the YNAP platforms in a consistent and accurate manner. Analytics and marketing tags and SDKs are managed through <a href="https://tagmanager.google.com/" target="_blank" title="Google Tag Manager">Google Tag Manager</a>, <a href="https://www.commandersact.com/en/solutions/tagcommander/" target="_blank" title="Tag Commander">Tag Commander</a> and <a href="https://firebase.google.com/" target="_blank" title="Firebase">Firebase</a>, through a combination of client-side and server-side deployments, always implemented to respect a user's consent. Proactive analysis and reporting is also performed on implementations and the performance of all Growth functions using a multitude of platforms including <a href="https://analytics.google.com/analytics/web/" target="_blank" title="Google Analytics">Google Analytics</a>, <a href="https://cloud.google.com/bigquery" target="_blank" title="BigQuery">BigQuery</a>, <a href="https://cloud.google.com/looker" target="_blank" title="Looker Studio">Looker Studio</a> and <a href="https://www.tableau.com/en-gb" target="_blank" title="Tableau">Tableau</a>, with the latter being developed to produce real-time and regular data quality reports, along with utilising Python scripts for anomaly detection.

Tags and SDKs implemented include <a href="https://analytics.google.com/analytics/web/" target="_blank" title="Google Analytics">Google Analytics</a>, <a href="https://www.criteo.com/" target="_blank" title="Criteo">Criteo</a>, <a href="https://www.adjust.com/" target="_blank" title="Adjust">Adjust</a>, <a href="https://contentsquare.com/" target="_blank" title="ContentSquare">ContentSquare</a>, <a href="https://www.qualtrics.com/uk/" target="_blank" title="Qualtrics">Qualtrics</a>, <a href="https://en-gb.facebook.com/business/help/952192354843755?id=1205376682832142" target="_blank" title="Facebook">Facebook (Meta)</a>, <a href="https://about.ads.microsoft.com/en-gb/solutions/tools/universal-event-tracking" target="_blank" title="Bing">Bing</a>, <a href="https://ads.tiktok.com/help/article?aid=9663" target="_blank" title="TikTok">TikTok</a>, <a href="https://rakuten.co.uk/" target="_blank" title="Rakuten">Rakuten</a> and <a href="https://partnerize.com/" target="_blank" title="Partnerize">Partnerize</a>, while the team is also the owner of <a href="https://marketingplatform.google.com/intl/en_uk/about/" target="_blank" title="Google Marketing Platform">Google Marketing Platform</a> and <a href="https://cloud.google.com/?hl=en" target="_blank" title="Google Cloud Platform">Google Cloud Platform</a>, with particular focus on <a href="https://cloud.google.com/bigquery" target="_blank" title="BigQuery">BigQuery and <a href="https://cloud.google.com/run" target="_blank" title="Cloud Run">Cloud Run. We are constantly researching the latest technologies and efficient implementation strategies as part of our ever-evolving roadmap, which has also resulted in the implementation of <a href="https://support.google.com/analytics/answer/10089681?hl=en" target="_blank" title="Google Analytics 4">Google Analytics 4</a>, <a href="https://developers.facebook.com/videos/2020/conversion-api-capi-overview/" target="_blank" title="Facebook Conversions API">Facebook Conversions API</a> and <a href="https://support.google.com/google-ads/answer/9888656?hl=en-GB" target="_blank" title="Google Enhanced Conversions">Google Enhanced Conversions</a>, along with server-side deployments of both <a href="https://www.commandersact.com/en/solutions/tagcommander/" target="_blank" title="Tag Commander">Tag Commander</a> and <a href="https://tagmanager.google.com/" target="_blank" title="Google Tag Manager">Google Tag Manager</a>. We have also developed an in-house Multi-Touch Attribution Model utilising <a href="https://support.google.com/analytics/answer/10089681?hl=en" target="_blank" title="Google Analytics 4">GA4</a> and <a href="https://cloud.google.com/bigquery" target="_blank" title="BigQuery">BigQuery</a>.
